# Vehicle

## Fields

### Type

The Type field defines the type of Vehicle.

Possible Options are:

- Truck
- Ship
- Train
- Airplane

### Vehicle Brand (Optional)

The Vehicle Brand field defines the brand of the vehicle.

### Tags (Optional)

The Tags field allowy you to define tags for this Vehicle.

It is useful to specify for example license plates for clearer definition.

### Notes (Optional)

The Notes field allows you to define any information to this Vehicle, which could be useful later on.
