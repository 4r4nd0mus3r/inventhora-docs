# Contact Person

## Fields

### First Name (Optional)

The FirstName field defines the First Name of the Contact Person.

### Last Name (Optional)

The Last Name field defines the Last Name of the Contact Person.

### Mobile Number (Optional)

The Mobile Number field defines the mobile phone number of the Contact Person.

### Phone Number (Optional)

The Phone Number field defines the phone number of the Contact Person.

### E-mail (Optional)

The E-mail field defines the e-mail address of the Contact Person.

### Schedule begins at (Optional)

The Schedule begins at field defines from which time on the Contact Person can be contacted.

### Schedule ends at (Optional)

The Schedule ends at field defines until which time the Contact Person can be contacted.

### Notes (Optional)

The Notes field allows you to define any information to this supplier, which could be useful later on.
