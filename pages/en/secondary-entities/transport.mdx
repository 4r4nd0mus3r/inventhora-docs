# Transport

## Fields

### Transport Agency (Optional)

The Transport Agency field defines the Transport Agency which is processing the transport.

See also: <Link href="/main-entities/transportagency">Transport Agency</Link>

### Vehicle (Optional)

The Vehicle field defines the vehicle which is processing the transport.

See also: <Link href="/secondary-entities/vehicle">Vehicle</Link>

### Price (Optional)

The Price field defines the cost for the transport.

### Date of Arrival (Optional)

The Date of Arrival field defines the date when the transport is arriving at it's destination.

### Arrival Point (Optional)

The Arrival Point field defines the location at which the transport is going to arrive.
