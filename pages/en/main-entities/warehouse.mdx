# Warehouse

## Fields

### Name

The Name field defines the name of the Warehouse

### E-mail (Optional)

The E-mail field defines the e-mail address of the Warehouse.

### Address (Optional)

The Address field defines the Address of the Warehouse.

This address appears on an <Link href="/invoice">Invoice</Link> directed to this Warehouse, if no billing address is specified.

Possible fields are:

- Address Line 1
- Address Line 2
- City
- State/Province
- Zip Code
- Country

### Mobile Number (Optional)

The Mobile Number field defines the mobile phone number of the Warehouse.

### Phone Number (Optional)

The Phone Number field defines the phone number of the Warehouse.

### Website (Optional)

The Website field defines the URL of the website of the Warehouse.

### Contact Person (Optional)

The Contact Person field defines the people who can be contacted at this Warehouse.

See also: <Link href="/secondary-entities/contactperson">Contact Person</Link>

### Notes (Optional)

The Notes field allows you to define any information to this Warehouse, which could be useful later on.

### Tags (Optional)

The Tags field allowy you to define tags for this Warehouse to categorize it.
