# User

## Fields

### Name

The Name field defines the name of the User.

### E-mail

The E-mail field defines the E-mail address of the User.

This E-mail is also going to be used by the user to access the account.

### Role

The Role field defines the role this user has. It allows you to restrict the permissions of the user.

See also: <Link href="/secondary-entities/roles">Roles</Link>
