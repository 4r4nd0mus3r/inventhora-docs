# Transport Agency

## Fields

### Name

The Name field defines the name of the Transport Agency

### E-mail (Optional)

The E-mail field defines the e-mail address of the Transport Agency.

### Address (Optional)

The Address field defines the Address of the Transport Agency.

Possible fields are:

- Address Line 1
- Address Line 2
- City
- State/Province
- Zip Code
- Country

### Billing Address (Optional)

The Billing Address field defines the Billing Address of the Transport Agency.

Possible fields are:

- Address Line 1
- Address Line 2
- City
- State/Province
- Zip Code
- Country

### Mobile Number (Optional)

The Mobile Number field defines the mobile phone number of the Transport Agency.

### Phone Number (Optional)

The Phone Number field defines the phone number of the Transport Agency.

### Website (Optional)

The Website field defines the URL of the website of the Transport Agency.

### Vehicles (Optional)

The Vehicles field defines the Vehicles which the Transport Agency owns/uses.

See also: <Link href="/secondary-entities/vehicle">Vehicle</Link>

### Contact Person (Optional)

The Contact Person field defines the people who can be contacted at this Transport Agency.

See also: <Link href="/secondary-entities/contactperson">Contact Person</Link>

### Notes (Optional)

The Notes field allows you to define any information to this Transport Agency, which could be useful later on.

### Tags (Optional)

The Tags field allowy you to define tags for this Transport Agency to categorize it.
